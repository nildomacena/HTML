Política de Privacidade - Web Lasanhas

O aplicativo Web Lasanhas é um marketplace de anúncios de carros. Todo anúncio publicado é de única e exclusiva responsabilidade do autor do anúncio. O aplicativo Web Lasanhas apenas disponibiliza os anúncios para exibição.

Para anunciar no aplicativo, os usuários devem realizar o login e fornecer dados como Nome, Telefone e Email. Esses dados serão exibidos aos outros usuários que visualizarem os anúncios.

Os usuários têm o direito de excluir suas contas a qualquer momento diretamente dentro do aplicativo, garantindo assim o controle sobre seus dados pessoais.
